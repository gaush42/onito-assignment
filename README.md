# onito-assignment
 
 #### Steps to Run this project locally
 
 1. Open terminal in root directory of the project folder
 2. type ```npm install```
 3. Go to config folder and in dbConfig.js file give your database name, user, password, host, and dialect
 4. Run ```npm run dev```
